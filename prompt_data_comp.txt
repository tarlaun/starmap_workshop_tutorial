You are a Geospatial Data Engineer specializing in Python and GeoPandas.

Your task is to generate a complete, executable Python script that performs
a spatial data composition workflow.

------------------------------------------------------------
STEP 1: INPUT DATASETS

You are given two GeoJSON datasets:

1. Parks dataset:
   - Contains polygon geometries representing parks
   - Attribute: PARK_NAME (string)

2. Vegetation dataset:
   - Contains polygon geometries representing vegetation types
   - Attribute: VEGDESC (string describing vegetation class)

------------------------------------------------------------
STEP 2: GOAL (Natural Language Task)

Compute the dominant vegetation type for each park.

Definition:
- For each park polygon, intersect it with all vegetation polygons
- Compute the area of overlap
- Assign the vegetation class (VEGDESC) with the largest overlap area
- If no overlap exists, assign "Unknown"

------------------------------------------------------------
STEP 3: GEOSPATIAL REQUIREMENTS

- Use GeoPandas
- Reproject both datasets to an equal-area CRS before computing areas
  (use EPSG:3310 — California Albers)
- Ensure geometries are valid (fix using buffer(0) if needed)
- Use spatial overlay (intersection)
- Compute area using geometry.area

------------------------------------------------------------
STEP 4: OUTPUT REQUIREMENTS

- Output must be a GeoJSON file
- Final CRS must be EPSG:4326 (WGS84)
- Output fields:
    - PARK_NAME
    - dominant_vegetation
    - geometry

------------------------------------------------------------
STEP 5: SCRIPT REQUIREMENTS

- The script must:
  - Accept 3 command-line arguments:
        <parks_geojson> <vegetation_geojson> <output_geojson>
  - Include clear structure:
        * helper function to load & clean data
        * function to compute dominant vegetation
        * main() function
  - Print useful progress logs
  - Be fully executable as a standalone script

------------------------------------------------------------
STEP 6: CODING STYLE

- Use clean, readable Python
- Use functions (no monolithic code)
- Add minimal helpful comments
- Avoid unnecessary dependencies (only geopandas, sys)

------------------------------------------------------------
FINAL OUTPUT FORMAT

Return ONLY the Python script.
Do NOT include explanations.
Do NOT include markdown fences.
